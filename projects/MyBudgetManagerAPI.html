MyBudgetManager API

An ASP.NET Web API project for managing budget-related data and user accounts.

Overview

The MyBudgetManager API is a RESTful backend service built with ASP.NET Web API. It handles budget management, user authentication, transactions, and categories, while securely providing integration with the mobile and desktop admin apps. The API enables seamless CRUD operations, ensuring smooth interactions across platforms.

Technologies Used

  • ASP.NET Web API
  • C#
  • Entity Framework Core
  • SQL Server
  • JWT Authentication
  • Data Annotations for Validation

Key Features

  • Secure user authentication using JWT tokens
  • CRUD operations for budgets, transactions, and categories
  • Data validation using Data Annotations
  • Integration with MyBudgetManager Mobile and Admin apps
  • Real-time data updates

Code Example


    // Example: API Controller for handling budget data
    [Authorize]
    [ApiController]
    [Route("api/[controller]")]
    public class BudgetsController : ControllerBase
    {
        private readonly IBudgetService _budgetService;
    
        public BudgetsController(IBudgetService budgetService)
        {
            _budgetService = budgetService;
        }
    
        [HttpGet]
        public async Task GetBudgets()
        {
            var budgets = await _budgetService.GetAllBudgetsAsync();
            return Ok(budgets);
        }
    
        [HttpPost]
        public async Task CreateBudget(BudgetDto budgetDto)
        {
            if (!ModelState.IsValid)
                return BadRequest(ModelState);
    
            var result = await _budgetService.CreateBudgetAsync(budgetDto);
            return CreatedAtAction(nameof(GetBudgets), new { id = result.Id }, result);
        }
    }
