MyBudgetManager API

ASP.NET Web API project for managing budget data.

Overview

The MyBudgetManager API is a RESTful API built with ASP.NET Web API. It serves as the backend for managing budget-related data, handling user accounts, transactions, and budget categories securely and efficiently. The API supports CRUD operations and integrates with both the mobile app and the admin desktop app.

Technologies Used

  • ASP.NET Web API
  • C#
  • Entity Framework Core
  • SQL Server
  • JWT Authentication
  • Data Annotations for Validation

Key Features

  • Secure user authentication using JWT tokens
  • CRUD operations for managing budgets, transactions, and categories
  • Data validation using Data Annotations
  • Integration with MyBudgetManager Mobile and Admin apps
  • Real-time data updates

Code Example


// Example: API Controller for handling budget data
[Authorize]
[ApiController]
[Route("api/[controller]")]
public class BudgetsController : ControllerBase
{
    private readonly IBudgetService _budgetService;

    public BudgetsController(IBudgetService budgetService)
    {
        _budgetService = budgetService;
    }

    [HttpGet]
    public async Task GetBudgets()
    {
        var budgets = await _budgetService.GetAllBudgetsAsync();
        return Ok(budgets);
    }

    [HttpPost]
    public async Task CreateBudget(BudgetDto budgetDto)
    {
        if (!ModelState.IsValid)
            return BadRequest(ModelState);

        var result = await _budgetService.CreateBudgetAsync(budgetDto);
        return CreatedAtAction(nameof(GetBudgets), new { id = result.Id }, result);
    }
}
