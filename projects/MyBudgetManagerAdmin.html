MyBudgetManager Admin App

A desktop admin app for managing budgets and user data, built with C#/WPF.

Overview

The MyBudgetManager Admin App is a desktop application built using Windows Presentation Foundation (WPF) and C#. It allows administrators to manage budgets, view user activities, and generate reports. This tool is essential for back-office tasks and ensures smooth financial operations.

Technologies Used

  • C#
  • WPF (Windows Presentation Foundation)
  • MVVM Architecture
  • SQL Server for Data Storage
  • REST API Integration (MyBudgetManager API)

Key Features

  • User management (CRUD operations)
  • Budget overview and detailed tracking
  • Real-time data synchronization with the MyBudgetManager API
  • Advanced reporting tools
  • Secure login and user roles management

Code Example


// Example: Fetching data from the MyBudgetManager API in C#
public async Task> FetchBudgetsAsync() {
    using (HttpClient client = new HttpClient()) {
        HttpResponseMessage response = await client.GetAsync("https://api.mybudgetmanager.com/budgets");
        if (response.IsSuccessStatusCode) {
            var json = await response.Content.ReadAsStringAsync();
            return JsonConvert.DeserializeObject>(json);
        }
        return null;
    }
}
