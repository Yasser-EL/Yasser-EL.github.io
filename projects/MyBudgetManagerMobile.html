MyBudgetManager Mobile App

A cross-platform mobile app for managing budgets, built with Flutter.

Under Construction: This project is currently being developed. Please check back soon for updates and the final release!

Overview

The MyBudgetManager Mobile App is built with Flutter, providing users with a seamless, cross-platform experience to manage budgets, track transactions, and set financial goals. Available for both Android and iOS, the app ensures real-time synchronization and supports offline mode for managing data locally.

Technologies Used

  • Flutter
  • Dart
  • REST API (ASP.NET Web API Backend)
  • Firebase for Authentication and Storage
  • SQLite for Local Data Storage

Key Features

  • Cross-platform (Android and iOS)
  • User authentication with Firebase
  • Real-time budget tracking and updates
  • Data synchronization with the MyBudgetManager API
  • Offline mode with local data storage using SQLite
