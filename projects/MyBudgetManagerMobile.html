MyBudgetManager Mobile App

A mobile app for managing your budget, built with Flutter.

Overview

The MyBudgetManager Mobile App is built using Flutter for cross-platform mobile development. It provides users with a seamless experience for managing their budgets, viewing transaction histories, and creating financial goals. The app is available on both Android and iOS platforms.

Technologies Used

  • Flutter
  • Dart
  • REST API (ASP.NET Web API Backend)
  • Firebase for Authentication and Storage
  • SQLite for Local Data Storage

Key Features

  • Cross-platform (Android and iOS)
  • User authentication with Firebase
  • Real-time budget tracking and updates
  • Data synchronization with the MyBudgetManager API
  • Offline mode with local data storage using SQLite
